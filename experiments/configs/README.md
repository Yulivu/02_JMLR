# Experiment Configs

Hand-written experiment intent lives here. These files should describe what to
run, not where temporary outputs are stored. Suite tasks route outputs under
`experiments/runs/`.

Current naming:

```text
exp1/  controlled and formal-validation tasks
exp2/  semi-real format tasks
exp3/  real-source small-field tasks
exp5/  WNUT17 BIO/NER R5 local smoke tasks
```
