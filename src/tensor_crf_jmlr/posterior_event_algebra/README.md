# Posterior Event Algebra

CPU-first reference implementation for finite CRF posterior-event algebra.

Scope:

- brute-force finite posterior event mass checks;
- complete DFA monitor utilities;
- product automaton transfer checks;
- indexing and order-`m` context boundary checks;
- tiny nonnegative MPO sanity constructors.

Non-claims:

- not an experiment runner;
- not a benchmark;
- not a production training system;
- not GPU code;
- not usefulness or JMLR-ready evidence.
